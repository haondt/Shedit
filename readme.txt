Shedit (no longer currently running but see /u/sheditbot)

A bot that slims down your reddit intake
usage -
python3 shedit.py [-s -c -cl]

shedit.py -> take 3 posts from every sub listed in subreddits.txt
shedit.py -s [subreddit] [num] -> take [num] posts from r/[subreddit]
shedit.py -c -> remove all posts on r/shedit
shedit.py -cl -> clear out the post log (log.txt)
